# AI-Powered Yoga App

## Quick Start

This Android app uses ML Kit to detect yoga poses in real-time. It currently supports:
- Mountain Pose
- Downward Dog  
- Warrior I

## Setup
1. Open in Android Studio
2. Build and run on a device with camera
3. Grant camera permissions
4. Point camera at person doing yoga poses

## Technical Stack
- **ML Kit Pose Detection**: Google's accurate pose detection
- **CameraX**: Modern Android camera API
- **Kotlin**: Primary development language
- **View Binding**: Type-safe view references

## Pose Detection
The app analyzes body landmarks to classify poses:
- Real-time camera analysis
- Confidence scoring
- Visual feedback in app UI
